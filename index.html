How to Get a RobotEvents API Token

You only need to do this once. The token can be reused indefinitely. The whole process takes about 5-10 minutes.

1

Sign Up / Log In to RobotEvents

If you don't have an account yet, register on the RobotEvents website. If you already have an account from registering for competitions, just log in.

https://www.robotevents.com/register
Tip: You can use the same account you use for VEX competition registration. No need to create a separate one.
2

Go to the API Page & Request Access

After logging in, visit the API documentation page:

https://www.robotevents.com/api/v2

Find the "Request Access" link or form on the page and fill in your details:

  • Name: Your name
  • Purpose: Explain why you need API access (see Step 3 for a template you can copy)

After submitting, some users get instant approval, while others may need to wait 1-3 days. You'll receive an email notification once approved.

3

Purpose Statement Template (Copy & Paste)

Paste the following into the purpose / description field of the request form:

I am requesting access to the RobotEvents API for the purpose of building a team scouting tool for the upcoming VEX V5 Robotics Competition (V5RC) World Championship. As a competing team, I would like to query team performance data including: - Skills rankings and scores (driver & programming) - Event results and qualification rankings - Win/loss records and award history This tool will be a local web application used solely for pre-match scouting and strategic preparation by our team members. I will fully respect the API rate limits and terms of service. Thank you for considering my request.
4

Once Approved, Generate Your Token

After approval, go back to the RobotEvents website and navigate to your account settings:

https://www.robotevents.com/api/v2/tokens

Or click your username in the top-right corner → Account Settings → scroll down to the API Tokens section.

Click the "Create New Token" button. A popup will appear where you can name and generate your token.

  • Token Name: Anything you like, e.g. vex-scout
  • Click Create, and a long string of text will appear — that's your token
Important: The token is only shown ONCE! Copy it immediately and paste it into the Token field in this tool. If you forget to copy it, you can delete it and create a new one.
5

Paste the Token & Start Scouting!

Paste the token into the "API Token" field at the top of this page, enter a team number, and hit Search.

The token is automatically saved in your browser's localStorage, so you won't need to re-enter it next time.

Tip: If the "Create New Token" button doesn't respond when clicked, this is a known bug on the RobotEvents website.
Try the following:
• Switch to a different browser (Chrome / Safari / Firefox / Edge)
• Use incognito / private browsing mode
• Clear your browser cache and try again
• Search or post for help on VEX Forum
